# BruvTab Privacy Policy

BruvTab is a browser extension that lets users control their browser tabs from the command line through a local native messaging host.

BruvTab does not collect, sell, share, or transmit user data to the developer or to any third-party service.

The extension may access browser tab information, including tab IDs, titles, URLs, active tab state, and window state, only to provide its tab-management features. When the user explicitly runs related CLI commands, BruvTab may also read page text, page HTML, words from a page, or capture the visible active tab screenshot.

This data is sent only between the browser extension and the locally installed BruvTab native messaging host on the user’s own machine. BruvTab does not send this data to any remote server.

BruvTab does not use analytics, advertising, tracking, or remote code.

Data handled by BruvTab is not retained by the extension, except as transient data needed to complete the user-requested command.
